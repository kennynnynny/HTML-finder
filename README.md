## ✨ Возможности

| Функция | Описание |
|---------|----------|
| 🔍 **Подсветка элемента** | Бирюзовая рамка вокруг элемента при наведении |
| 📋 **Копирование HTML** | 4 формата: `outerHTML`, `innerHTML`, атрибуты, классы |
| 📐 **Box Model** | Визуальные зоны margin/padding/content прямо на странице |
| 📏 **Измерение расстояний** | Alt + наведение показывает линию между элементами |
| 🧭 **Навигация по DOM** | Ctrl + ↑/↓ или Ctrl + колесико для перемещения по дереву |
| ⚙️ **Popup-настройки** | Компактный интерфейс без лишних кнопок |
| 🎨 **Авто-сохранение** | Настройки применяются мгновенно |

---

## 🎯 Использование

| Действие | Результат |
|----------|-----------|
| **Навести курсор** | Подсветка элемента + панель с размером и классами |
| **Ctrl/Alt/Shift + Click** | Копирование HTML в буфер обмена |
| **Ctrl + ↑ / ↓** | Навигация: родитель → ребёнок |
| **Ctrl + колесико ↑/↓** | То же, через мышь |
| **Alt + зажать и вести** | Измерение расстояния между элементами |
| **Курсор к краю экрана** | Расстояние от элемента до края |
