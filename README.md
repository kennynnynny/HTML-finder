# 📘 HTML Copy on Hover

Расширение для быстрого копирования HTML-разметки элементов веб-страниц с помощью наведения и `Ctrl+Click`.

## ✨ Возможности

- 🔍 **Визуальная подсветка** элемента при наведении курсора
- 📋 **Копирование HTML** в буфер обмена по `Ctrl+Click`
- 💬 **Toast-уведомления** об успехе/ошибке
- 🛡️ **Безопасность** — не влияет на интерактив страницы
- ⚡ **Производительность** — минимальное влияние на CPU/Memory

## 🚀 Установка

### Локальная разработка (Chrome/Edge/Opera/Brave)

1. Откройте браузер и перейдите:
   - **Chrome**: `chrome://extensions`
   - **Edge**: `edge://extensions`
   - **Opera**: `opera://extensions`

2. Включите **"Режим разработчика"** (переключатель в правом верхнем углу)

3. Нажмите **"Загрузить распакованное"**

4. Выберите папку `html-copy-hover` из этого репозитория

5. ✅ Расширение установлено! Откройте любую страницу и наведите курсор на элемент

### Firefox

1. Откройте `about:debugging#/runtime/this-firefox`
2. Нажмите **"Загрузить временное дополнение"**
3. Выберите файл `manifest.json` из папки `html-copy-hover`
4. ✅ Расширение работает!

## 📖 Использование

| Действие | Результат |
|----------|-----------|
| **Навести курсор** | Подсветка элемента синей рамкой |
| **Ctrl + Click** | Копирование `outerHTML` элемента в буфер |
| **Уведомление** | Toast "✅ HTML скопирован" появляется на 2 секунды |
